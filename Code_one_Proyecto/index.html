"CODE_ONE"


Code_one es un fantástica herramienta web donde poner a prueba nuestro código HTML, CSS y Javascript de forma fácil y cómoda.

Su manejo es sencillo, en la columna de la izquierda podemos seleccionar las dependencias y opciones de configuración que necesitamos para nuestras pruebas. La zona de la derecha esta dividida en cuarto apartados, uno para introducir código HTML, otro para el código CSS, otro para el código JavaScrip
